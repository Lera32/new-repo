Редакторы

Существует множество различных редакторов для работы с кодом. Разные редакторы имеют плюсы и минусы в работе.

Я выбрала Visual Studio Code. Мне очень нравится этот редактор. Основной причиной выбора среды Visual Studio для создания данного программного продукта была исключительно высокая простота и скорость разработки прикладных решений. Продукты Microsoft для разработчиков давно входят в список наиболее востребованного программного обеспечения для программистов разного уровня. За восемь лет существования на рынке среда разработки Microsoft Visual Studio стала де-факто стандартом создания .NET-приложений.

Другие редакторы не так популярны как этот. Brackets не настолько удобен для меня, потмоу что мой основной опыт работы в Visual Studio Code.

Плюсы Visyal Studio Code

Вот некоторые плюсы этой программы
  1. Кроссплатформенность. Работает на MacOS, Ubuntu и Windows.
  2. Бесплатность. Простой open-source редактор и плагины — платить не надо.
  3. Легковесность. Потребуется совсем мало ресурсов, чтобы приступить к работе с минималистичным VSC.

Минусы Visyal Studio Code

  1. Программа тяжеловесная, запустится далеко не на любом компьютере, работает медленнее других редакторов.
  2. Интерфейс перегружен, надо разбираться со всеми дополнительными функциями.
  3. Не работает на Linux.
Итогом я хочу сказать, что выбор редактора - это личный выбор каждого пользователя. Лично для меня эта программа знакома и понятна. Её функционал для начала пути достаточен. Не отрицаю, что в будущем редактор поменяется, но пока этот выбор я считаю рациональным для себя.
